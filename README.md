# TermuxSetup
The simple script for setup termux

# Requirements
  - Android Device installed termux
  - Internet / Wifi
  - Brain.exe

# How to use
  - pkg update && pkg upgrade -y && pkg install -y git wget
  - git clone https://github.com/Error404-TH/TermuxSetup.git
  - cd TermuxSetup
  - chmod +x setup && ./setup or bash setup
